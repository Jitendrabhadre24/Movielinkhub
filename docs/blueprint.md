# **App Name**: MovieLink Hub

## Core Features:

- Firebase Email Authentication: Secure user signup and login using only email and password via Firebase Authentication.
- Content Browsing for Guests: Allow all users, including unauthenticated guests, to freely browse the entire content catalog (movies, TV shows, anime, genres).
- Personalized Watchlist Management: Authenticated users can add or remove items from their personalized watchlist, persisted in Firestore.
- Continue Watching Feature: Authenticated users can track and resume viewing content from where they left off, with progress stored in Firestore.
- Dynamic Content Integration (TMDB API): Seamlessly fetch and display an extensive catalog of movies, TV shows, and anime details from the TMDB API, organized by genres and individual content pages.
- User Account Page: A dedicated section where authenticated users can view their email, manage their watchlist, access 'Continue Watching' items, and log out.
- Intuitive Bottom Navigation: Provide a fixed bottom navigation bar for quick access to Home, Search, Genres, and Account sections, enhancing mobile usability.

## Style Guidelines:

- The primary color is a rich gold (#F4C025), selected to evoke a sense of premium entertainment and quality, contrasting brilliantly against the dark theme.
- The background color is a very deep, muted brown-black (#15130F), providing a sleek, dark aesthetic that minimizes eye strain and allows content to stand out.
- The accent color is a warm, desaturated terracotta (#B8502E), which provides a harmonious contrast with the gold, suitable for interactive elements and subtle highlights.
- The 'Inter' font (sans-serif) is used throughout for both headlines and body text, providing a clean, modern, and highly legible experience across all content.
- Utilize a consistent set of minimal, clean outline icons for navigation and actions, ensuring clarity and maintaining the elegant, Netflix-style aesthetic.
- Implement a Netflix-style content grid and smooth-scrolling horizontal rows to present movies, TV shows, and anime, enhancing visual engagement and discoverability.
- A responsive, sticky bottom navigation bar is essential for an optimal user experience on various device sizes, ensuring key app sections are always accessible.
- Incorporate subtle, performant transitions and hover effects for interactive elements and content rows to provide a fluid, premium feel without hindering speed.
